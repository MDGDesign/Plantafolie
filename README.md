# Plantafolie

School project
Objective: An MVC Web site using ASP.NET CORE 2.0.
           In the web site there are some functionality to establish:
           
           Site functionality:
              - Customer accounts
              - Login/Logout
              - Shopping carts
              - Oders
              - Management of products
              - Customer management
              - Basic stats
              
           Page's functionality:
              - Search
              - Pagination
              - Sorting
              - Categories
